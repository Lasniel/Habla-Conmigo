🛠️ Requisitos técnicos

Para asegurar el correcto funcionamiento del sitio y una experiencia de aprendizaje óptima, se recomienda lo siguiente:

Navegador y dispositivo

  • ✔ Navegadores recomendados: Chrome, Firefox, Edge o Safari (versiones recientes)
  • ✔ Resolución mínima de pantalla: 1024×768 px
  • ✔ Compatible con tablets y dispositivos móviles

Requisitos de audio

  • ✔ Altavoces o auriculares funcionales
  • ✔ Se recomienda un entorno sin ruido para las actividades de repetición

Conectividad y sistema

  • ✔ Conexión a Internet estable (mínimo 1 Mbps)
  • ✔ Sistema operativo: Windows 10+, Android 8+, iOS 12+, macOS 10.13+
  • ✔ JavaScript habilitado en el navegador

Accesibilidad

  • ✔ Compatible con lectores de pantalla y navegación por teclado
  • ✔ Imágenes con texto alternativo (`alt`) descriptivo
  • ✔ Colores y contrastes aptos para todo tipo de usuarios

💡 Si encuentras dificultades técnicas, asegúrate de actualizar tu navegador o probar en otro dispositivo.
